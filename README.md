# Complete Vehicle Simulator

## Project Overview

A parametric, OEM-grade digital vehicle simulatopr built around a configurable ICE powertrain. Validated against real OBD-II data. Architecture covers ECU software, engine & powertrain modeling, vehicle dynamics, CAN networking and diagnostics.
